Chat

You must authenticate before joining a chat. You can join ANY chat you wish. The bot will perform all of its tasks regardless of which chat you are in. However, if it is not a mod in the chat some abilities will be limited.

The bot is still a user, it can be banned if you are causing problems. This feature is intented so mods can add bots to the streamers chat if the streamer does not want to install the bot themselves.

This is an experimental feature. It may be removed in a future update.

Select Join Chat = >

Enter the chat that you want to join.

If everything is successful the bot will join the selected chat. Chat controls will unlock on the right.

When you are finished disconnect the bot. It cleanly closes the connection.
