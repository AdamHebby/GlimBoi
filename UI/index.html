Welcome to GlimBoi

To get started please authenicate with Glimesh. You need a Developer Application for GlimBoi to function properly. This allows the bot to have a custom name.

Click here to create an application. It takes less than 5 minutes and gives the bot full functionality. When you are finished, click the button below to authenicate the bot with your application.

GlimBoi is an open source project started by Mytho. It is written in Electron which is a framework for building desktop applications. Server side is NodeJS. Data storage is NEDB. The front end is HTML and Bootstrap CSS. DataTables are used for showing the database and Apex charts for that chart =>

It was inspired by Scottybot (Rip Mixer). Special thanks to Daddy Robot and Codestics for the help and inspiration. The goal of this bot is to provide an easy to use interface while using minimal resources in server mode.

Contributions are always welcome :)
